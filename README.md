## Leandecks Webpack Template

Webpack template using Babel and Sass.   
Source code in `./src`, builds to `./dist`
.
### Commands

Run for development, opens local dev server and watches for changes:

`
npm run dev
`

Build all resources from `./src` to `./dist`:

`
npm run build
`
